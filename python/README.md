This package contains the Python code needed for preprocessing and analyzing data related to the agent-based model developed in [PAPER]. This code has been designed mostly for internal use and is being made public in order to facilitate reproducability of the various analyses and figures generated as part of the aforementioned publication. 

The various functions included in the package have been are sorted into analysis, animation, and preprocessing, and visualization categories:
* `analysis.py` includes code necessary for processing the ABM outputs, extracting timeseries data, calculating neighborhoods, etc.;
* `animation.py` provides the streamlined code necessary for creating animations of the ABM's spatiotemporal outputs give an output directory;
* `preprocessing.py` includes code necessary for assigning cell type labels from multiplexed immunohistochemistry data either from a list of rules or via random forest; additionally, includes code to resize spatial images as needed based on target cell densities;
* `sensitivity_analysis.py` includes code used specifically to analyze and plot things related to sensitivity analysis, since these typically require accessing larger amounts of ABM outputs with different structures relative to normal outputs
* `visualization.py` includes code necessary for visualizing the data generated by the ABM;

Additionally, `constants.py` includes static definitions for the colormap used for plotting and other variables referenced by the various submodules.
